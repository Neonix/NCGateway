NCGateway
=================

[DEMO]()

## Require

- composer.phar
- git
- php > 5.3
    

## Linux OSX
    
1. git clone https://github.com/Neonix/NCGateway.git
2. composer install 
3. php bin/console doctrine:database:create
4. php bin/console doctrine:schema:update -force
5. php bin/console server:run
6. open http://127.0.0.1:8000
